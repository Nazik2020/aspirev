# AI Engineer Roadmap

## What is an AI Engineer?

An AI Engineer uses pre-trained models and existing AI tools to improve user
experiences. They focus on applying AI in practical ways, without building
models from scratch. This is different from AI Researchers and ML Engineers,
who focus more on creating new models or developing AI theory.

## Stages Overview

### Stage 1 — Introduction to AI (Beginner)
Core concepts: What is an AI Engineer, LLMs, Inference, Training, Embeddings,
Vector Databases, RAG, Prompt Engineering, AI Agents, Common Terminology.

### Stage 2 — Pre-trained Models & APIs (Beginner)
OpenAI, Claude, Gemini, Azure AI, AWS Sagemaker, Hugging Face, Mistral,
Cohere, Replicate. Learn capabilities, context length, and knowledge cut-offs.

### Stage 3 — OpenAI Platform (Intermediate)
Chat Completions API, Writing Prompts, Token Management, Pricing,
OpenAI Playground, Fine-tuning.

### Stage 4 — AI Safety & Ethics (Intermediate)
Prompt Injection, Security, Bias & Fairness, Moderation API,
Adversarial Testing, Safety Best Practices.

### Stage 5 — Open Source AI (Intermediate)
Hugging Face Hub, Open Source Models, Inference SDK,
Transformers.js, Ollama.

### Stage 6 — Embeddings & Vector Databases (Intermediate)
OpenAI Embeddings, Sentence Transformers, Semantic Search,
Chroma, Pinecone, Weaviate, FAISS, LanceDB, Qdrant, Supabase, MongoDB Atlas.

### Stage 7 — RAG & Implementation (Advanced)
RAG Pipeline (Chunking → Embedding → Vector DB → Retrieval → Generation),
LangChain, LlamaIndex, OpenAI Assistant API.

### Stage 8 — AI Agents (Advanced)
Agent Usecases, ReAct Prompting, OpenAI Functions/Tools,
Manual Implementation, OpenAI Assistant API.

### Stage 9 — Multimodal AI (Advanced)
Vision API, DALL-E, Whisper, Image/Audio/Video Understanding,
LangChain & LlamaIndex for Multimodal Apps.

### Stage 10 — Development Tools (Advanced)
AI Code Editors (Cursor, Copilot), Code Completion Tools.
